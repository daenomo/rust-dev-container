# rust-dev-container
A Docker development container for Rust projects with pre-configured tools and dependencies.

## Getting Started

```bash
git clone git@github.com:daenomo/rust-dev-container.git
cd rust-dev-container
docker compose up -d 
```

## Features

- Rust toolchain
- Cargo package manager
- Common development tools

## Requirements

- Docker
- Docker Compose (optional)
- vscode (devcontainer)
